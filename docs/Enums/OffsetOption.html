OffsetOption

public enum OffsetOption
extension OffsetOption: Codable
extension OffsetOption: Hashable

Indicates where in the context playback should start. See PlaybackRequest.

Only available when contextURI is an album or playlist (not an artist) or when uris([SpotifyURIConvertible]) is used for the context. One of the following:

  • position(Int): The index of the item in the context at which to start playback.
  • uri(SpotifyURIConvertible): The URI of the item to start playback at.
