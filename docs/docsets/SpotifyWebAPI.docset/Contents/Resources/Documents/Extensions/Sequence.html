Available where Element == SpotifyImage

  • largest

    Returns the largest image in this sequence of SpotifyImage, or nil if the sequence is empty.

    When determining the largest image, Images with nil for height and/or width are considered to have a height and/or width of 0. The largest image is calculated based on height * width.
