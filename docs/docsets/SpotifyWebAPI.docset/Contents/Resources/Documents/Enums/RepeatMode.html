RepeatMode

public enum RepeatMode : String, Codable, Hashable, CaseIterable

The repeat mode of the user’s player. Either off, track, or context.

  • off

    Repeat mode is off.

    Declaration

    Swift

    case off
  • track

    The current track is playing on repeat.

    Declaration

    Swift

    case track
  • context

    The current context, such as a playlist, is playing on repeat.

    Indicates that when the user reaches the end of the context, the track/episode at the beginning of the context will play again, and so on. If repeat mode is off, then, depending on the user’s settings, playback may stop after the user reaches the end of the context or songs similar to the context may begin to play.
