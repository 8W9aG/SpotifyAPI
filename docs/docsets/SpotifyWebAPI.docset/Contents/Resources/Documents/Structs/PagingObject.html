PagingObject

public struct PagingObject<Item> : Paginated where Item : Decodable, Item : Encodable, Item : Hashable
extension PagingObject: Codable
extension PagingObject: Hashable

A Spotify paging object.

The offset-based paging object is a container for a set of objects. It contains a key called items (whose value is an array of the requested objects) along with other keys like previous, next and limit that can be useful in future calls.

  • href

    A link to the Spotify web API endpoint returning the full result of the request in this PagingObject.

    Use SpotifyAPI.getFromHref(_:responseType:), passing in the type of this PagingObject to retrieve the results.

Convienence Methods

  • estimatedTotalPages

    The total number of pages available, including this page.

    This property is calculated by dividing total by limit and rounding up to the nearest integer. For example, if total is 745 and limit is 100, then estimatedTotalPages is 8.

    • total represents the maximum number of items available to return.

    • limit represents the maximum number of items in this page (as set in the query or by default).

    Warning

    This calculation assumes that the limit for each page will be the same as this page.
