Spotify Dates

  • decodeSpotifyDate(forKey:)

    Decodes a Date from a date-string with one of the following formats:

    • “YYYY-MM-DD”
    • “YYYY-MM”

    • “YYYY”

    Throws

    If the value cannot be decded into a string, or if the format of the string does not match any of the above formats.

    Declaration

    Swift

    func decodeSpotifyDate(forKey key: Key) throws -> Date

    Parameters

    key

    The key that is associated with a date string in one of the above formats.

    Return Value

    The decoded Date.
